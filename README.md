This is a comprehensive group project where we analyzed Spanish electoral data from 2008 to 2019. Using **R**, we processed multiple complex datasets to understand the behavior of political parties, polling accuracy, and regional voting patterns.

The project was built using **Quarto**, allowing us to combine code, analysis, and interactive visualizations in a single document.

### ğŸ› ï¸ What we did

* **Data Cleaning (Tidyverse):** Extensive use of `dplyr` and `tidyr` to filter time periods, handle missing values, and standardize political party names.
* **Polling Analysis:** Compared pre-election surveys with actual results to see which media outlets and pollsters were more accurate for parties like PP and PSOE.
* **Visualization:** Created complex charts with `ggplot2` and used the `mapSpain` library to generate electoral maps.
